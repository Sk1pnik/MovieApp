## Tech Stack
* REST API - Retrofit
* Dependency injection - Hilt
* Database - Room
* Pagination - Paging  3
* Key-value pair data storage - Preferences DataStore
*  Kotlin Coroutines
* Kotlin Flow
* Navigation Component
* View Binding
* Glide
